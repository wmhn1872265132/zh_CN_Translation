# MathCAT 快捷键表述调整

本次仅调整 `Translation/user_docs/userGuide.xliff` 中 MathCAT 数学导航段落的快捷键表述：

- 将裸写的快捷键补为反引号格式：`esc 键` -> `Esc`，`Ctrl+回车` -> `Ctrl+回车`。
- 将快速开始说明中的 `行首`、`行尾`、`空格`、`Shift+上光标`、`Shift+下光标`、`Esc` 等写法统一为反引号快捷键，不在反引号后追加“键”。
- 将数学导航命令表“按键”列中的具体快捷键统一为裸快捷键 token，如 `左光标`、`Ctrl+左光标`、`回车`、`空格`、`行首`、`行尾`、`退格`。
- 将典型用法段落中的 `退格`、`右光标`、`左光标`、`Shift+上光标`、`Shift+下光标` 去掉反引号后的“键”。
- 保留泛称写法 `光标键`、`左右光标键`、`下光标键`、`Ctrl+光标键`，未改为“方向键”。
